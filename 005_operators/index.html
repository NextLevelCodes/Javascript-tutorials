Operators in javascript

1.Arithmetic Operators

+ Addition

- Subtraction

* Multiplication

/ Division

% Modulus (Remainder)

** Exponentiation (Power)

2. Assignment Operators

= , Example: x=2 , Equivalent to: x = 2

+= , Example: x += 2, Equivalent to: x = x + 2

-= , Example: x -= 2 , Equivalent to: x = x - 2

*= , Example: x *= 2 , Equivalent to: x = x * 2

/= , Example: x /= 2 , Equivalent to: x = x / 2

%= , Example: x %= 2 , Equivalent to: x = x % 2

**= , Example: x **= 2 , Equivalent to: x = x ** 2

3. Logical Operators

&& (logical and)

|| (logical or)

! (logical not)

4. Bitwise Operators
