Increment and Dcrement Operators

Operator Type Syntax Description Example
++ Increment (Postfix) x++ Returns current value, then increments let a = 5;
console.log(a++); // 5
console.log(a); // 6
++ Increment (Prefix) ++x Increments first, then returns new value let b = 5;
console.log(++b); // 6
-- Decrement (Postfix) x-- Returns current value, then decrements let c = 10;
console.log(c--); // 10
console.log(c); // 9
-- Decrement (Prefix) --x Decrements first, then returns new value let d = 10;
console.log(--d); // 9
